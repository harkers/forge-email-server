Priority Scoring Formula

score = (severity × 3) + (blocking × 3) + (breadth × 2) + (deadline × 2) + readiness - effort

Safety Gates

Auto-dispatch is blocked when:

  1. Destructive operations — unless allowDestructive: true
  2. Production impact — unless allowProdChanges: true
  3. Token ceiling exceeded — estimate > maxTokensPerJob
  4. Failed dependencies — prerequisite in quarantine
  5. Missing metadata — incomplete task definition
  6. Low confidence — readiness below threshold
