# DevForge: Project Foundry
## MVP Roadmap

Status: Planning  
Date: 2026-03-28

## MVP Principle

Build the smallest complete governed loop first.

The MVP should prove that an idea can be shaped, documented, governed, orchestrated, reviewed, risk-tracked, communicated, and prepared for enablement without collapsing into ad-hoc delivery.

## MVP Scope

### Must prove
- controlled project intake
- architecture and control framing
- governance gate before build
- canonical documentation path
- controlled orchestration foundation
- pipeline state and traceability model
- planning and execution handoff
- review and fix loop
- risk register and mitigation history foundation
- comms distillation outputs
- training readiness outputs

## Planned Sprint 0

- define root structure
- define forge-* workspace domains
- define minimal schemas
- define minimal templates
- define bootstrap and validation approach
- define ownership and CODEOWNERS model

## Planned Sprint 1 Epics

### E1 — Forge Orchestrate v1
Create the starter orchestration foundation and repo scaffolding approach.

### E2 — ForgePipeline v1
Define status model, workflow state basics, and stub integration path.

### E3 — Forge Document Engine v1
Define document structures, core records, and evidence linkage expectations.

### E4 — ForgeComms v1
Define distillation, handoff pack generation, and audience-facing summary rules.

### E5 — ForgeTraining v1
Define training input truth, impact classification, persona tailoring foundation, and Product Readiness Pack structure.

### E6 — ForgeRisk v1
Define risk register format, evaluation cycle, mitigation log, and update triggers.

## MVP Exit Criteria

- planning pack approved
- repository design agreed
- skeleton and MVP epics agreed
- one pilot-ready template structure defined
- risk, privacy, comms, and training are explicit rather than implied

## Post-MVP Candidates

- Persona Tailor v2
- Impact Analyzer v2
- stronger pipeline integration
- visual asset automation
- control mapping to standards
- richer DLP and metadata enforcement
