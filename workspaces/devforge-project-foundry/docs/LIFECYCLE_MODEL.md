## Lifecycle Logic

- Primary work stages produce outputs.
- ForgeComms converts raw outputs into review-ready, stakeholder-ready, and handoff-ready artifacts.
- ForgeRisk runs as a continuous risk cycle and is revisited on new information, major changes, failures, and mitigations.
- ForgeTraining produces role-based enablement assets from approved delivery truth rather than draft assumptions.
- Forge Fix routes failed work back into the governed flow without losing history.

## Principle

Define → Distil → Design → Govern → Record → Orchestrate → Plan → Build → Review → Release → Validate → Train → Communicate → Close
