# Display Forge

A fresh workspace for a polished visual dashboard focused on projects, tasks, and notes.

## Included

- Fancy static web dashboard starter
- Sample project/task data
- Notes area for planning and decisions
- Lightweight local server script

## Layout

- `app/public/` — frontend assets
- `app/data/` — dashboard data
- `notes/` — product and design notes
- `docs/` — supporting docs
- `scripts/` — helper scripts

## Repository Structure

- `apps/admin-ui/` — admin frontend
- `apps/player-ui/` — signage/player frontend
- `services/api/` — backend API service
- `services/worker/` — RSS polling and background jobs
- `packages/shared/` — shared types/helpers
- `infra/docker/` — container/runtime definitions
- `infra/scripts/` — ops/startup scripts
- `config/` — config templates
- `storage/` — media/cache/logs/backups
- `tests/` — test suites
