# Display Forge

A fresh workspace for a polished visual dashboard focused on projects, tasks, and notes.

## Included

- Fancy static web dashboard starter
- Sample project/task data
- Notes area for planning and decisions
- Lightweight local server script

## Layout

- `app/public/` — frontend assets
- `app/data/` — dashboard data
- `notes/` — product and design notes
- `docs/` — supporting docs
- `scripts/` — helper scripts
