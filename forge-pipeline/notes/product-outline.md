# Product Outline

## Working title

Forge Pipeline

## Core idea

A project-organised task and notes workspace with a visually pleasing dashboard.

## Key entities

- Project
- Task
- Note
- Activity

## Must-have features

- Create and edit projects
- Add tasks under projects
- Track status, priority, due dates
- Store extensive notes per project
- Filter tasks by status / project / tag
- Nice web presentation

## Nice-to-have later

- Drag-and-drop board
- Search
- Markdown notes
- Export / backup
- Timeline or activity feed
