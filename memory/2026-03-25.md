## Control-plane-what-next v2 upgrade

**Task:** Incorporate v2 specification pack into existing skill

**Merged from:** `control-plane-what-next-v2-pack.zip`

**New features:**

### Deterministic Priority Scoring
Formula: `(severity*3) + (blocking*3) + (breadth*2) + (deadline*2) + readiness - effort`
- Bands: P0 (18+), P1 (13-17), P2 (8-12), P3 (0-7)

### Safety Gates (auto-dispatch stops when)
1. Destructive operations (unless `allowDestructive: true`)
2. Production changes (unless `allowProdChanges: true`)
3. Secrets/privileges handling
4. Missing metadata
5. Invalid approval window
6. Token ceiling exceeded
7. Failed/quarantined dependency
8. Low confidence
9. Conflicting concurrency

### State Schema
File: `~/.openclaw/workspace/.control-plane-what-next-state.json`
- `activeWindow`: approval window state
- `session`: completed/failed/quarantined jobs
- `lifetime`: cumulative statistics
- `current`: last dispatch, in-flight job
- `policy`: destructive/prod flags, token limits

### Tie-Break Rules
1. Higher blocking impact
2. Earlier due date
3. Higher execution readiness
4. Lower token cost
5. Older queue age
6. Lexical ID

### References Added
- `03-state-schema.json`: full JSON schema
- `04-priority-scoring-model.md`: scoring formula
- `05-operational-guardrails.md`: stop conditions
- `06-operator-output-examples.md`: example outputs
- `07-implementation-outline.md`: implementation approach
- `08-test-plan.md`: test scenarios
