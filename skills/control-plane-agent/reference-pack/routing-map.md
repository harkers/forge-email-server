# Routing Map

## Suggested default routing

- Privacy incident, containment, risk, breach summaries -> `privacy-incident`
- Vendor due diligence, transfer review, hosting/access location extraction -> `vendor-assessor`
- External source gathering, standards lookup, public policy checks -> `research`
- Email/report/briefing polishing -> `drafting`
- Merge, quality gate, reassignment -> `manager`

## Review rules

Always route back to `manager` before:
- sending email
- generating final client-facing report
- writing records into an authoritative system
- triggering downstream actions
