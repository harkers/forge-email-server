# Reflection Template

Use this after meaningful work.

```text
CONTEXT: [task type]
INTENT: [what success should have looked like]
OUTCOME: [what actually happened]
GAP: [difference between intent and outcome]
LESSON: [what to do differently next time]
ACTION: [none | log to corrections.md | log to domains/<domain>.md | log to projects/<project>.md]
```

## Notes

- Keep entries brief
- One lesson per reflection
- Prefer concrete future behavior over abstract commentary
- If there is no reusable lesson, set ACTION to `none`
