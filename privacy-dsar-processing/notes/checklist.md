# DSAR Processing Checklist

- Log request receipt date
- Assign case ID
- Confirm deadline and jurisdiction
- Verify identity if needed
- Clarify scope if request is broad
- Identify systems and custodians
- Preserve evidence / originals
- Search and collect data
- Review for privilege / exemptions / third-party data
- Prepare redactions
- Draft response
- Record what was disclosed and why
- Close case with audit notes
