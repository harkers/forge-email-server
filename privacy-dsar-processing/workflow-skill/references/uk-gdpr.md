# UK GDPR DSAR Guidance

## Deadline rules

- Default response period: one calendar month from receipt.
- If the same date does not exist in the following month, use that month's final calendar day.
- Extend by up to two further months only where the request is complex or numerous.
- Notify the requester about the extension within the original one-month window.

## Suggested case tracking fields

Record:

- date received
- standard deadline
- extension invoked
- extension notice date
- extended deadline
- identity check dates
- clarification dates
- final response date

## Recommended templates

Use workspace templates for:

- acknowledgement
- clarification request
- extension notice
- refusal or limited response

## Caution

UK GDPR timing and exemptions can be fact-sensitive. Use legal review for edge cases, contested identity issues, mixed third-party records, or refusal decisions.
