## Documentation and registration requirements

This task is not complete until:
- implementation is finished
- validation is complete
- rollback is documented
- TOOLS.md is updated with the delivered capability

Required TOOLS.md updates:
- tool/service name
- project code
- purpose
- runtime location
- ports/bindings/sockets
- authentication method
- storage paths
- dependencies
- operational commands
- validation commands
- rollback steps
- security notes
- related services / impact

## Definition of done

A task is only complete when:
- implementation is complete
- validation is complete
- rollback is documented
- TOOLS.md is updated
- AGENTS.md is updated if the task introduces a new standing rule
